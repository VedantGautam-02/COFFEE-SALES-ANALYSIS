# ☕ Coffee Sales Analysis

An end-to-end Data Analytics project that analyzes coffee shop sales using **SQL**, **Python**, and **Power BI** to discover sales trends, customer behavior, and business insights.

---

## 📌 Project Overview

This project focuses on analyzing coffee shop sales data to understand business performance and generate meaningful insights.

The project covers:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Business Analysis
- Python Data Visualization
- Interactive Power BI Dashboard
- Business Insights & Recommendations

---

## 🎯 Objectives

- Analyze monthly sales performance
- Identify peak sales hours
- Find best-selling products
- Compare store performance
- Generate actionable business insights
